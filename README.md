# LLM Evaluation Portfolio

## Goal
This project is a personal portfolio showcasing a structured manual evaluation process comparing single-turn responses from three frontier Large Language Models:
- **ChatGPT**
- **Claude**
- **DeepSeek**

The objective is to evaluate 12 realistic, practical prompts using a consistent methodology, demonstrating structured thinking, comparative analysis, and qualitative reasoning.

## Scope
This portfolio evaluates only single-turn responses. Multi-turn follow-up behavior is outside the scope of this project.

## Repository Structure
- `evaluations/` — Contains 12 completed prompt evaluation files (`P01` to `P12`), including the raw responses and qualitative analysis.
- `rubric.md` — Detailed scoring criteria and evaluation philosophy.
- `README.md` — This file.

## Method & Criteria
1. Use the same prompt for all three models.
2. Read and evaluate the initial responses.
3. Score them on a scale of 1-5 across five consistent criteria:
   - **Helpfulness**: Practical value and relevance.
   - **Clarity**: Readability and structure.
   - **Practicality**: How actionable the advice is.
   - **Coverage**: Completeness in addressing the prompt.
   - **Overall Quality**: Synthesis of overall performance.
4. Write a short comparative analysis highlighting strengths, weaknesses, key differences, and a final ranking.

## Limitations
- One evaluator
- Single-turn only
- Scores reflect practical, qualitative judgment rather than rigorous statistical benchmark testing
