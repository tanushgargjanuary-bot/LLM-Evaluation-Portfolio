# LLM Evaluation Rubric

## Scoring Scale

1 = Very Poor
2 = Poor
3 = Acceptable
4 = Good
5 = Excellent

---

## 1. Instruction Following

Did the model answer what was actually asked?

Score:
1–5

## 2. Accuracy

Score:
1–5

Score:
